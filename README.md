# 🌤️ Weather Forecast Webpage

## 📌 Project Overview

The **Weather Forecast Webpage** is a responsive and visually engaging web application built using **React JS** and **Tailwind CSS**. It displays real-time weather information for a selected city, including temperature, humidity, wind speed, and weather conditions. The application integrates the **OpenWeatherMap API** to fetch live weather data.

The design focuses on:

* Clean and modern UI
* Smooth CSS animations
* Responsive layout across all devices
* Interactive and user-friendly interface

Background visuals (videos and images) are sourced from:

* 🎥 Videos: [https://pixabay.com/videos/](https://pixabay.com/videos/)
* 🖼️ Images: [https://www.pexels.com/](https://www.pexels.com/)

## ✨ Features

### 🌡️ 1. Weather Display Section

Displays current weather details for a selected city:

* Temperature
* Weather condition (Sunny, Cloudy, Rainy, etc.)
* Humidity
* Wind speed
* Weather condition icons

Smooth animations are applied when weather data loads or updates.

---

### 📅 2. Forecast Section

Includes a 5-day forecast with:

* Date
* Minimum temperature
* Maximum temperature
* Weather condition summary

Forecast cards include hover effects and animated transitions.

---

### 🎨 3. CSS Animations

To enhance user experience, the application includes:

* Animated weather icons (e.g., rotating sun, moving clouds, falling rain effects)
* Smooth transitions when displaying weather data
* Hover effects on forecast cards

---

### 🔍 4. Interactive Features

* Search bar to select a city
* Weather data updates dynamically based on search
* Optional refresh button to reload weather data

---

### 📱 5. Responsive Design

The layout is fully responsive and works seamlessly on:

* Mobile devices
* Tablets
* Laptops
* Desktop screens

Tailwind CSS responsive utilities are used to ensure adaptability across screen sizes.

## 🔑 API Configuration

This project uses the **OpenWeatherMap API**.

### Step 1: Get an API Key

1. Create an account at [https://openweathermap.org/](https://openweathermap.org/)
2. Generate your API key from the dashboard

### Step 2: Add Environment Variable

Create a `.env` file in the root directory:

```
VITE_WEATHER_API_KEY=your_api_key_here
```

⚠️ Do not upload your `.env` file to GitHub.
